1. Background

The heart of memory research is to precisely state the storage and retrieval mechanism. The storage mechanism is called an engram.

Definition 1.0.1 (Engram)

The enduring change in the brain that an experience produces.

What we’ve learned is that Engrams are typically a sparse subset of neurons. Investigation in CA3 pyramidal neurons shows that these while these neurons are initially typical, they rapidly become more excitable over the course of a day past when the memory was made. More precisely, there’s a lot more eletrical inputs that trigger action potentials, and a lot more prolonged spiking activity. Interestingly, there’s also a lot more inhibitory inputs. This kind of makes sense, for the engram to be useful we need it activate if and only if a reasonable context arises. I’m not sure why this can’t be achieved with just excitatory inputs, but perhaps those cannot create a precise enough context by themselves.

1.1. Sidenote

Prolonged spiking activity is interesting, because it implies a single input can cause a neuron to spike many times, and indeed this is true! The ODE for the voltage across neurons can be modeled as an resistor-capacitor circuit with time-varying resistors and ion pumps as a forcing function. The forcing function plus linear term provided by the resistors allows for an oscillating steady-state: repeated spiking. This model also explains how an inhibitory signal works. If the ion channels suddenly have lower resistance, then the voltage across all the spiking channel will drop, and so the likelyhoods of action potentials will drop.
